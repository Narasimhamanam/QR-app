# 🚀 QR Code Generator App

A simple and powerful QR Code Generator built using **HTML, CSS, and JavaScript** with customization and login features.

---

## ✨ Features

- 🔐 Fake Login System (Frontend only)
- ⚡ Real-time QR Code Generation
- 🎨 Custom QR Color
- 🖼️ Add Logo inside QR
- 📥 Download QR Code (PNG)
- 💻 Clean and Responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- QR Code Styling Library
